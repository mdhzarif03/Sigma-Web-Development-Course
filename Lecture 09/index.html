ID AND CLASSES IN HTML

In HTML, we can use the id and class attributes to identify and group elements for styling and scripting purposes.

ID Attribute

The id attribute is used to uniquely identify an element on a page. Each element can have only one ID, and it must be unique within the document.

<div id="header">
    <h1>Welcome to My Website</h1>
</div>

Class Attribute

The class attribute is used to group elements for styling and scripting purposes. An element can have multiple classes, separated by spaces.

<div class="container">
    <h1 class="title">Welcome to My Website</h1>
</div>

In this example, the div element has a class of "container", and the h1 element has a class of "title". We can use CSS to style these elements based on their classes.
