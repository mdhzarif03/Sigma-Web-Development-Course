Note: This is a simplified guide to CSS implementation in HTML. For more detailed information, refer to the official CSS documentation.

There are three ways to add CSS to HTML:

  1. Inline CSS
  2. Internal CSS
  3. External CSS

1. Inline CSS

Inline CSS is added directly to an HTML element using the "style" attribute tag <style="property: value;">. For example:

<p style="color: red;">This text is red.</p>

This text is red.

2. Internal CSS

Internal CSS is added within a <style> tag in the <head> section of the HTML document. For example: 

<style>
.blue-text {
  color: blue;
}
</style>

This text is blue.

3. External CSS

External CSS is added by linking to an external stylesheet using the <link> tag in the <head> section of the HTML document. For example:

In the <head> section:

<link rel="stylesheet" href="styles.css">

In the "styles.css" file:

.green-text { color: green; }

This text is green.

When to Use Each Method

Inline CSS is useful for quick styling or when you want to override existing styles.

Internal CSS is good for small projects or when you want to keep all styles in one file.

External CSS is recommended for larger projects as it allows for better organization and reusability of styles.

Summary

CSS can be added to HTML using inline styles, internal stylesheets, or external stylesheets. Each method has its own use cases and advantages.

In practice, it's common to use a combination of these methods depending on the specific needs of the project.

For best practices, it's generally recommended to use external CSS for maintainability and separation of concerns.
