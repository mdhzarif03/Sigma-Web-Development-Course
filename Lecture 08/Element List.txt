Comprehensive List of Block and Inline Elements (Most commonly used)

Block

<div> — generic container (most used)
<h1> — main heading
<h2> — section heading
<h3> — subsection heading
<h4> — smaller heading
<h5> — smaller heading
<h6> — smallest heading
<p> — paragraph
<section> — section of content
<article> — self-contained content (blog/news)
<header> — page/section header
<footer> — page/section footer
<main> — main content area
<nav> — navigation links
<aside> — sidebar content
<ul> — unordered list
<ol> — ordered list
<li> — list item
<table> — table
<thead> — table head
<tbody> — table body
<tfoot> — table footer
<tr> — table row
<td> — table cell
<th> — table header cell
<form> — form container
<fieldset> — form grouping
<legend> — form group title
<address> — contact info
<blockquote> — long quotation
<pre> — preformatted text
<hr> — horizontal rule
<figure> — media grouping
<figcaption> — caption for figure
