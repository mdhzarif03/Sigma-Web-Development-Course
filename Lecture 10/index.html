Media Elements

In this lecture, we will learn about media elements in HTML, including the <video>, <audio>, and <iframe> elements. These elements allow you to embed multimedia content such as videos, audio files, and other web pages into your HTML documents.

Video Element

The <video> element allows you to embed video content in your web page. You can specify the source of the video using the src attribute, and you can also include controls for play, pause, and volume using the controls attribute. The loop attribute allows the video to play continuously, and the poster attribute specifies an image to be shown while the video is downloading or until the user hits the play button.

Audio Element

The <audio> element allows you to embed audio content in your web page. You can specify the source of the audio using the src attribute, and you can also include controls for play, pause, and volume using the controls attribute. The loop attribute allows the audio to play continuously.

Audio element with controls and loop.

IFrame Element

The <iframe> element allows you to embed another HTML document within the current document. This is commonly used to embed content from other websites, such as videos from YouTube or maps from Google Maps. The src attribute specifies the URL of the page to embed, and you can also specify the width and height of the iframe.
