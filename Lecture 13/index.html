HTML Entities

HTML entities are used to display reserved characters in HTML. For example, the less than symbol (<) is represented as &lt; and the greater than symbol (>) is represented as &gt;.

Here are some common HTML entities:

  • &lt; - less than
  • &gt; - greater than
  • &amp; - ampersand
  • &quot; - double quote
  • &apos; - single quote
  • &nbsp; - non-breaking space
  • ' - apostrophe
  • © - copyright
  • ® - registered trademark

Using HTML entities ensures that your content is displayed correctly and prevents issues with reserved characters in HTML.

For more information on HTML entities, refer to the W3Schools HTML Entities page.
