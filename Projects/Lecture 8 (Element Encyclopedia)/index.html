Element Encyclopedia

Element
A fundamental building block of HTML, representing a part of the document.
Elements type
  • Block elements
  • Inline elements
  • Void elements

Block Elements

Block elements are those that start on a new line and take up the full width available. Examples include:

  • <div>
  • <h1> to <h6>
  • <p>
  • <ul>
  • <ol>

Inline Elements

Inline elements do not start on a new line and only take up as much width as necessary. Examples include:

  • <span>
  • <a>
  • <strong>
  • <em>

Void Elements

Void elements are self-closing and do not have any content. Examples include:

  • <img>
  • <br>
  • <hr>
  • <input>
